SAÉ 23

Quel est l'objectif de ce site ?

Ce site a pour objectif d'exploiter les données provenant de capteurs répartis dans les bâtiments de l’IUT en proposant une interface dédiée.
De plus pour ce faire nous avons utiliser des projtes comme le "traitement de données".

Mais qu'est-ce que la SAE23 ?

Ce projet a pour objetif d'offrir aux gestionnaires des bâtiments de l’IUT une interface conviviale et simple pour la visualisation des données capteur.
L'objectif principal va être de proposer une solution de visualisation de données publiées sur un bus MQTT dans une IHM accessible via un navigateur web
Pour réussir ce projet nous avons donc utiliser plusieurs outils :
  • phpMyadmin pour la base de données
  • Grafana pour visualiser les données
  • Node-RED pour la programmation

Que va présenter ce site web ?

Ce site web va permettre d'afficher les différentes parties du projet :
  • Une page administration
  • Une page gestion
  • Une page consultation
  • Une page pour nous contacter
  • Les mentions légales de notre site web
